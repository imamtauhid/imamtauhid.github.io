Octopress Di Github

Setelah sekitar (kurang lebih) dua hari untuk men-deploy blog ini akhirnya berhasil online. Octopress ini memang berbeda dengan Wordpress, Joomla, Tumblr, dan sejenisnya karena butuh kemauan lebih untuk mengerti kode program, jadi wajar jargon nya “A Blogging Framework for Hackers”.

Octopress merupakan framework yang dibuat dan di design oleh Brandon Mathis untuk Jekyll, Apa itu Jekyll ?? Jekyll adalah engine static site generator sederhana untuk web personal, sebuah project, atau suatu website organisasi. Jekyll merupakan mesin yang berkerja dibalik Github Pages sehingga kita dapat meng-hosting sebuah website dari repo Github.
